# Project 01

## Term Deposit Subscription Prediction Using Machine Learning

### Objective

Develop a machine learning classification model to predict whether a bank customer will subscribe to a term deposit based on demographic, financial, and marketing campaign information.

### Project Highlights

- Data Cleaning and Preprocessing
- Feature Engineering
- Binary Encoding
- One-Hot Encoding
- Exploratory Data Analysis (EDA)
- Data Visualization
- Logistic Regression
- Random Forest Classifier
- Model Evaluation
- ROC Curve Analysis
- SHAP Explainable AI (XAI)

### Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- SHAP
- Jupyter Notebook
