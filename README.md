# Project-01

## Term Deposit Subscription Prediction Using Machine Learning

### Objective

Develop a machine learning classification model to predict whether a bank customer will subscribe to a term deposit based on demographic, financial, and marketing campaign information.

### Project Highlights

- Data Cleaning and Preprocessing
- Feature Engineering
- Binary Encoding
- One-Hot Encoding
- Exploratory Data Analysis (EDA)
- Data Visualization
- Logistic Regression
- Random Forest Classifier
- Model Evaluation
- ROC Curve Analysis
- SHAP Explainable AI (XAI)

### Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SHAP
- Jupyter Notebook

# How to Use

## 1. Clone the Repository

```bash
git clone https://github.com/huzaifawaheed2/DevelopersHub-Corporation-Advanced-Internship.git
```

---

## 2. Navigate to a Project Folder

### Project-01

```bash
cd DevelopersHub-Corporation-Advanced-Internship/Project-01-Term-Deposit-Prediction
```

### Project-02

```bash
cd DevelopersHub-Corporation-Advanced-Internship/Project-02-Customer-Segmentation
```

---

## 3. Install Required Libraries

```bash
pip install -r requirements.txt
```

---

## 4. Open Jupyter Notebook

```bash
jupyter notebook
```

---

## 5. Run the Notebook

### Project-01

```text
notebooks/term_deposit_prediction.ipynb
```

### Project-02

```text
notebooks/Customer_Segmentation.ipynb
```
